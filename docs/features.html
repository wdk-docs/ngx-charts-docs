特征¶

开箱即用的，我们要创建在其上构建图表低水平健壮的框架。下面的图表类型外:

图表类型¶

  • 水平和垂直条形图（标准，分组堆积，标准化）

  • 线

  • 区（标准，堆积，标准化）

  • 馅饼（可展开，网格，自传说）

  • 甜甜圈

  • 规

  • 线性仪表

  • 力向图 (已过时 - 请用 ngx-graph 代替)

  • 热图

  • 树形图

  • 号码牌

  • 泡泡/散

定制¶

  • 自动缩放

  • 时间轴滤波

  • 直线插补

  • 配置轴标签

  • 传奇（标签和梯度）

  • 高级标签定位

  • 实时数据支持

  • 先进的工具提示

  • 响应式布局

  • 数据点事件处理程序

  • 与升级工程

  • 启用/禁用动画

如果你有一个新功能的想法，创建一个Github上的问题，甚至更好PR）。 确保包括使用情况下，该图的一个例子的图像（如果适用）和数据格式。

备择方案¶

图表框架是一角钱一打今天。没有多少人存在，几年前，当我们在内部开始这个项目，其中许多人还没有真正有角工作。 这是我们的努力过程中编译开源这个最近的列表中，我们:

  • ng2-nvd3 - 使用Angular2组件nvd3开源包装器

  • ng2-charts - 开源Angular2包装指令的chart.js之

  • angular2-highcharts - 开源Angular2包装器HighCharts

  • recharts - 开源组合的图表库建立在反应的组分

  • Vega - 开源的帆布/ SVG即框架

  • C3 - 基于D3开源可重复使用的图表库

  • Plotly - 商业商业智能和数据科学。

  • Highcharts - 商业图表框架

  • eCharts - 开源的JavaScript图表框架

  • dc.js - 开源的JavaScript图表库
