介绍¶

声明图表框架Angular2和超越！

NGX-图表是独一无二的，因为我们不只是换D3，也没有任何其他的图表引擎对这一问题。 它是使用角来渲染和动画与所有它的结合和速度善良的SVG元素，和用途D3为优良的数学函数，体重秤，轴和形状发电机等 具有角做所有的渲染它打开我们到无限可能的角平台提供了诸如AOT，通用等。

数据可视化是一门科学，但是，这并不意味着它必须是丑陋的。 其中一个我们在创建这个项目所做的大力度是使图表美观。 该款式也通过CSS完全可定制的，所以你可以重写他们为你请。

此外，构建自定义图表通过利用该通过NGX-图表模块露出的各种NGX-图表组件是可能的。

点击这里 结帐交互式演示或更多信息检查出图表的实例在左侧。
