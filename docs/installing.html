安装¶

您可以从 发布页面 获取最新版本在Github上或通过NPM。

  • npm install @swimlane/ngx-charts --save

此外，发行代码签入和居住在 这里.

检查 package.json 文件角NGX-图表的版本是兼容的。

JavaScript¶

  • release/index.js - UMD版

  • release/index.min.js - UMD版精缩

发展¶

如果你想在本地运行演示，只是做:

  • npm i

  • npm start

  • 浏览 http://localhost:9999
